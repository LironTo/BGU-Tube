# 🎓 BGU-Tube (Moodle Records Downloader)

ברוכים הבאים ל-**BGU-Tube**!
מערכת חכמה וידידותית להורדת הקלטות וידאו מאתר ה-Moodle של אוניברסיטת בן-גוריון (BGU) בקליק אחד.

המערכת פותחה כדי לחסוך לסטודנטים זמן יקר, ומאפשרת הורדה מרוכזת של הרצאות ותרגולים ישירות למחשב, כשהן מסודרות לפי קורסים.

---

## ✨ פיצ'רים מרכזיים

- 🚀 **התחברות אוטומטית:** התחברות קלה ומהירה למודל באמצעות אוטומציה (Playwright).
- 📚 **שליפת קורסים חכמה:** סריקת כל הקורסים אליהם אתם רשומים בסמסטר הנוכחי.
- 👨‍🏫 **סינון לפי מרצה/מתרגל:** בחירת ההקלטות להורדה לא רק לפי קורס, אלא גם לפי מי שהעלה אותן (מרצה או מתרגל ספציפי).
- ⚡ **הורדה יעילה:** שימוש בהורדה בזרם (Streaming) דרך `httpx` שמונעת קריסות (OOM) גם בהורדת קבצים כבדים במיוחד (GBs).
- 🖥️ **ממשק משתמש (GUI):** אפליקציית חלונות נוחה וקלה לשימוש, ללא צורך בידע מקדים בשורת הפקודה.

---

## 🛠️ דרישות מערכת ותלויות

המערכת כתובה ב-Python ומשתמשת במספר ספריות חיצוניות.
כדי להריץ את המערכת, יש לוודא שמותקן אצלכם **Python 3.8** ומעלה.

התקנת הספריות הדרושות:
```bash
pip install playwright nest_asyncio yt-dlp aiohttp aiofiles httpx pillow
```
לאחר התקנת Playwright, חובה להתקין את הדפדפנים שלו:
```bash
playwright install chromium
```

## 🚀 איך משתמשים?

1. **שיבוט המאגר (Clone):**
   ```bash
   git clone https://github.com/yourusername/BGU-Tube.git
   cd BGU-Tube
   ```

2. **הרצת המערכת:**
   ```bash
   python src/UI.py
   ```

3. **התחברות:**
   - הזינו את שם המשתמש והסיסמה שלכם לאוניברסיטה בממשק.
   - *טיפ למקצוענים:* ניתן ליצור קובץ בשם `LoginInfo.py` תחת תיקיית `Media/` ובו להגדיר:
     ```python
     USERNAME = "my_username"
     PASSWORD = "my_password"
     ```
     כך הפרטים יטענו אוטומטית בכל פעם (הקובץ כמובן ב-`.gitignore` ולא יעלה לרשת).

4. **בחירת קורסים והורדה:**
   - סמנו ב-✔ את הקורסים מהם תרצו להוריד הקלטות.
   - לחצו על "עבור לבחירת מרצים".
   - בחרו מאילו מרצים/מתרגלים להוריד (יופיע גם מספר ההקלטות הזמינות לכל אחד).
   - לחצו על "התחל הורדה".

📁 ההקלטות יישמרו באופן מסודר בתוך תיקיית `downloads/`, תחת תיקייה נפרדת לכל קורס, עם שם ההרצאה ושם המרצה!
