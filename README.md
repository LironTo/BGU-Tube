# BGU-Tube

כלי קטן שכתבתי כדי להוריד הקלטות מהמודל של אוניברסיטת בן-גוריון. במקום להוריד וידאו אחרי וידאו ידנית, אפשר פשוט להתחבר פעם אחת, לסמן את הקורסים והמרצים שרוצים, והכלי יוריד את הכל בצורה מסודרת לתוך תיקיות.

הסיבה העיקרית שכתבתי את זה היא שהמודל עושה בעיות לפעמים עם קבצים כבדים (OOM), אז האפליקציה פה מורידה את הוידאו ב-chunks (streaming) ומונעת את הקריסות האלה.

## מה זה עושה בגדול?
- מתחבר אוטומטית למודל (משתמש ב-Playwright ברקע).
- מוצא את כל הקורסים שלכם.
- נותן לכם לבחור ממי בדיוק בא לכם להוריד את ההקלטות (מרצה/מתרגל).
- מוריד עד 3 סרטונים במקביל ושומר הכל תחת תיקיית `downloads`.

## דרישות
חובה שיהיה מותקן פייתון (3.8 לפחות).
קודם כל נתקין את כל הספריות הדרושות דרך קובץ ה-requirements:
```bash
pip install -r requirements.txt
```

חשוב גם להתקין את הדפדפן של Playwright (אחרת ההתחברות למודל לא תעבוד):
```bash
python -m playwright install chromium
```

## איך מריצים
1. מורידים את הקוד:
```bash
git clone https://github.com/LironTo/BGU-Tube.git
cd BGU-Tube
```
2. מריצים את הסקריפט:
```bash
python src/UI.py
```

**טיפ קטן להמשך:**
אם אין לכם כוח להכניס יוזר וסיסמה כל פעם מחדש, אפשר פשוט ליצור קובץ בשם `LoginInfo.py` בתוך התיקייה `Media/` ולשים בו:
```python
USERNAME = "my_username"
PASSWORD = "my_password"
```
אל תדאגו, הקובץ מוגדר ב-`.gitignore` אז הסיסמה שלכם לא תעלה בטעות לגיטהאב.
