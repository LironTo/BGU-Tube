# BGU-Tube

כלי שכתבתי כדי להוריד הקלטות מהמודל של אוניברסיטת בן גוריון. לאפשר את האופציה להוריד את כל ההקלטות (לפי מרצה ומתרגל) בקורס מסויים. מתאים מאוד לאנשי מילואים שמשרתים במקום ללא אינטרנט שמאפשר צפייה בשידורים.

## תהליך העבודה

הממשק מנחה אתכם ב-4 שלבים:

1. **התחברות** — כניסה עם שם משתמש וסיסמה של BGU.
2. **בחירת קורסים** — הכלי שולף אוטומטית את כל הקורסים שלכם ומציג אותם לבחירה.
3. **בחירת מרצים** — לכל קורס שנבחר, הכלי מציג את רשימת המרצים/מתרגלים ואת מספר ההקלטות שלהם.
4. **הורדה** — אישור אחרון ולחיצה על "התחל הורדה". הקבצים נשמרים תחת `downloads/<שם הקורס>/`.

עד 3 הורדות רצות במקביל כדי לחסוך זמן.

## דרישות

נדרש Python 3.8 ומעלה.

```bash
pip install -r requirements.txt
```

לאחר מכן יש להתקין את דפדפן Playwright (חובה להתחברות):

```bash
python -m playwright install chromium
```

## הרצה

```bash
git clone https://github.com/LironTo/BGU-Tube.git
cd BGU-Tube
python src/UI.py
```

## שמירת פרטי כניסה

אם אין חשק להכניס שם משתמש וסיסמה בכל הפעלה, אפשר ליצור קובץ `Media/LoginInfo.py`:

```python
USERNAME = "my_username"
PASSWORD = "my_password"
```

הקובץ מוגדר ב-`.gitignore` ולא יעלה לגיטהאב.
